flexbox properties always given to the parent element

flexbox has --->
1. display:flex;
2.flex-direction:row(default), column, row-reverse, column-reverse;
3.flex-wrap:nowrap(default), wrap, wrap-reverse;
4. flex-flow: column wrap; {{means it is a shorthand of flex-direction and flex-wrap properties}} flex-flow: flex-direction flex-wrap;
5.justify-content: flex-start,flex-end, start, end ,center, space-between, space-around, space-evenly; [[ALIGN ITEMS WITH RESPECT TO THE X-AXIS]]
6.align-items:stretch(default), flex-start, flex-end, center; [[ALIGN ITEMS WITH RESPECT TO THE Y AXIS]]
7. align-content:flex-start, flex-end, start, end, center, space-between, space-around, space-evenly;
[[ALIGN ITEMS WITH RESPECT TO THE Y-AXIS BUT IT ONLY WORKS WHEN IT HAS MULTILINE ITEMS AND FLEX-WRAP IS WRAP OR WRAP-REVERSE]]
8. gap -- px; same as margin to the ITEMS
9. row-gap: px; give margin b/w the rows
10. column-gap: px; give margin b/w the columns
